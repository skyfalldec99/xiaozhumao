本计算模块使用办法

1.安装相关库包括 Feon，Meshio，numpy  ，Pyqt， easygui ，matplotlib

2.运行UI.py

3.按照提示，打开已安装的gmsh.exe，Mesh之后选择show node labels，输入相关荷载信息

注意：以空格为间隔输入多组数据

4.选择保存节点和单元的文件夹

5.选择GMSH生成的.msh文档

6.点击计算

7.选择绘图

此处可在show.py中进行修改相关参数，输出不同的应力，位移云图

nodes.txt格式解析

例

0 0.0000 0.0000 0 0
1 1.0000 0.0000 0 0
2 0.0000 1.0000 0 0
3 0.0000 0.9000 0 0
4 0.0000 0.8000 0 0
5 0.0000 0.7000 0 0
6 0.0000 0.6000 0 0
7 0.0000 0.5000 0 0
8 0.0000 0.4000 0 0
9 0.0000 0.3000 0 0

- 列0：节点标识符（int）

- 列1：x坐标（float）

- 列2：y坐标（float）

- eles.txt解析

  0 3 0 33 34 81
  1 3 0 76 64 87
  2 3 0 74 36 76
  3 3 0 44 33 81
  4 3 0 74 76 87
  5 3 0 36 74 75
  6 3 0 79 68 86
  7 3 0 45 83 104
  8 3 0 47 80 81
  9 3 0 83 48 104
  10 3 0 64 76 85

  

  `eles.txt`包含元素信息。文件中的每一行都定义单个元素的信息，并由以下字段组成：

  - 列0：元素标识符（整数）。

  - 第1栏：元素类型（整数）：

    - 4节点四边形为1。
    - 2为6节点三角形。
    - 3表示一个三节点三角形。

  - 第2列：当前元素（整数）的材料属性。

  - 第3列至结尾：元素列表，这是符合每个元素的节点的列表。节点应以逆时针方向列出。
